Panduan Ganti WiFi

Berikut adalah panduan langkah demi langkah untuk mengganti nama (SSID) dan password WiFi untuk beberapa model modem yang umum digunakan di jaringan kami.

Pastikan Anda memilih panduan yang sesuai dengan model modem WiFi yang Anda gunakan di rumah. Informasi model modem biasanya tertera pada stiker di bagian bawah atau belakang perangkat.

TOTOLINK N200RE (V.4 & V.5)

  1. Hubungkan perangkat Anda (komputer atau HP) ke modem TOTOLINK, baik melalui kabel LAN atau WiFi.
  2. Buka browser (Chrome, Firefox, dll.) dan ketik alamat IP 192.168.0.1 atau 192.168.1.1 di address bar, lalu tekan Enter.
  3. Anda akan diminta login. Masukkan username admin dan password admin atau admin123.
  4. Setelah berhasil login, masuk ke menu "Wireless".
  5. Untuk mengganti nama WiFi, ubah isian pada kolom "SSID".
  6. Untuk mengganti password, cari kolom "Password" dan masukkan password baru Anda.
  7. Klik "Apply" untuk menyimpan perubahan. Modem akan restart secara otomatis.

TOTOLINK N355RT

  1. Hubungkan perangkat ke modem N355RT.
  2. Buka browser dan masukkan alamat IP 192.168.0.1 atau 192.168.1.1.
  3. Login dengan username admin dan password admin atau admin123.
  4. Untuk mengganti nama WiFi, ubah isian pada kolom "SSID".
  5. Untuk mengganti password, cari kolom "Password" dan masukkan password baru Anda.
  6. Klik "Apply" untuk menerapkan perubahan.

TP-LINK TL-WR820N

  1. Hubungkan perangkat ke modem TL-WR820N.
  2. Buka browser dan ketik tplinkwifi.net atau 192.168.0.1.
  3. Jika ini pertama kali, Anda akan diminta membuat password login baru. Jika sudah pernah, masukkan password login yang telah Anda buat. Defaultnya adalah admin.
  4. Setelah login, masuk ke menu "Wireless" atau "Nirkabel".
  5. Ubah nama WiFi pada bagian "Nama Jaringan (SSID)".
  6. Ubah password pada bagian "Password" atau "Kata Sandi".
  7. Klik "Save" atau "Simpan". Modem akan restart.

TP-LINK TL-WR840N

  1. Hubungkan perangkat ke modem TL-WR840N.
  2. Buka browser dan ketik tplinkwifi.net atau 192.168.0.1.
  3. Login dengan username admin dan password admin atau admin123.
  4. Masuk ke menu "Wireless", lalu pilih "Wireless Security".
  5. Ubah password Anda di kolom "Wireless Password".
  6. Klik "Save" untuk menyimpan perubahan.
  7. Masuk ke menu "Wireless", lalu pilih "Basic Settings".
  8. Untuk mengubah nama WiFi, ubah isian pada kolom "Wireless Network Name (SSID)".
  9. klik "Save" untuk menyimpan perubahan.

TENDA N301

  1. Hubungkan perangkat ke modem TENDA N301.
  2. Buka browser dan ketik 192.168.0.1.
  3. Login dengan password admin atau admin123 (username biasanya tidak diminta).
  4. Masuk ke menu "Wireless Settings".
  5. Ubah nama WiFi di kolom "WiFi Name (SSID)".
  6. Ubah password di kolom "WiFi Password".
  7. Klik "OK". Modem akan menyimpan pengaturan.

Mengatasi Masalah Umum

Tidak bisa terhubung ke WiFi setelah ganti password

Ini adalah masalah paling umum. Perangkat Anda (HP/Laptop) masih menyimpan password lama. Solusinya:

  1. Pada perangkat Anda, cari daftar jaringan WiFi yang tersedia.
  2. Klik pada nama WiFi Anda, lalu pilih opsi "Lupakan Jaringan" (Forget Network).
  3. Cari kembali jaringan WiFi Anda dan sambungkan lagi dengan memasukkan password yang baru.

Tidak bisa membuka halaman login modem

  1. Pastikan perangkat Anda sudah terhubung ke modem yang benar (baik via kabel atau WiFi).
  2. Pastikan alamat IP yang Anda masukkan di browser sudah benar (misal: 192.168.0.1). Coba alamat alternatif seperti 192.168.1.1.
  3. Coba gunakan browser yang berbeda atau clear cache browser Anda.
  4. Jika semua cara gagal, langkah terakhir adalah melakukan reset pabrik pada modem, namun ini akan mengembalikan semua pengaturan ke awal, termasuk pengaturan dari ISP. Hubungi kami sebelum melakukan ini.

Koneksi internet lambat atau sering terputus

  1. Coba matikan modem selama 1-2 menit, lalu nyalakan kembali.
  2. Pindahkan posisi modem ke tempat yang lebih terbuka dan minim halangan (seperti tembok tebal atau perabotan besar).
  3. Jauhkan modem dari perangkat elektronik lain yang dapat menyebabkan interferensi, seperti microwave atau telepon nirkabel.
  4. Jika masalah berlanjut, hubungi tim support kami untuk pengecekan lebih lanjut.
